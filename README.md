# 🚗 Jonatha Mecânica e Ar Condicionado

Projeto desenvolvido como trabalho freelancer para a empresa **Jonatha Mecânica e Ar Condicionado Automotivo**, com foco em presença digital, apresentação de serviços e captação de clientes via WhatsApp.

## 📌 Sobre o Projeto

O site foi desenvolvido para fortalecer a marca no ambiente digital, apresentar os serviços oferecidos e facilitar o contato direto com clientes da região.

O layout foi pensado para transmitir:

- Profissionalismo  
- Confiança  
- Clareza nas informações  
- Facilidade de navegação  

## 🛠️ Tecnologias Utilizadas

- HTML5 (estrutura semântica)
- CSS3 (layout responsivo e animações)
- JavaScript (interações e funcionalidades)
- Bootstrap (componentes auxiliares)
- Google Maps Embed
- Deploy em hospedagem própria

## ⚙️ Funcionalidades Implementadas

- Menu hamburger responsivo
- Slider automático de imagens
- Animações suaves com CSS
- Botão flutuante de WhatsApp
- Integração com Google Maps
- Layout adaptado para Desktop, Tablet e Mobile
- Scroll suave entre seções

## 📱 Responsividade

O projeto foi desenvolvido utilizando abordagem responsiva com Media Queries, garantindo boa experiência em:

- 📱 Smartphones
- 💻 Notebooks
- 🖥️ Desktop

## 🎯 Objetivo do Cliente

- Aumentar a visibilidade local
- Facilitar o contato via WhatsApp
- Apresentar serviços e diferenciais
- Reforçar credibilidade da marca
